🥇 1. 什么 vs 哪 САМАЯ ВАЖНАЯ
❓ БЕЗ ОГРАНИЧЕНИЙ
什么 shénme что / какой
Спрашиваем содержание, название, свойство — вариантов бесконечно много.
你喜欢什么书？ Nǐ xǐhuān shénme shū?
📌 Порядок: 你 + 喜欢 + 什么 + 书？
🎯 ВЫБОР ИЗ НЕСКОЛЬКИХ
哪 nǎ какой из
Спрашиваем конкретный выбор из ограниченного числа. ОБЯЗАТЕЛЬНО со счётным словом!
哪本书是你的？ Nǎ běn shū shì nǐ de?
📌 Порядок: 哪 + 本 + 书 + 是 + 你的？
💡 Лайфхак: Если можно добавить «из этих» → 哪. Если просто «что за?» → 什么.
2. 怎么 vs 怎么样 vs 什么样
🔧 СПОСОБ ДЕЙСТВИЯ
怎么 zěnme как
Спрашиваем как сделать / каким образом.
你怎么去学校？ Nǐ zěnme qù xuéxiào?
📌 Порядок: 你 + 怎么 + 去 + 学校？
📊 СОСТОЯНИЕ / ОЦЕНКА
怎么样 zěnmeyàng как дела
Спрашиваем состояние, качество, оценку.
你怎么样？ Nǐ zěnmeyàng?
📌 Порядок: 你 + 怎么样？
👤 ВНЕШНОСТЬ / ВИД
什么样 shénme yàng как выглядит
Спрашиваем внешность, форму, тип.
他什么样？ Tā shénme yàng?
📌 Порядок: 他 + 什么样？
3. 谁 vs 什么人 vs 哪个人
👤 ИМЯ / ЛИЧНОСТЬ
谁 shéi кто
Спрашиваем имя или личность человека.
他是谁？ Tā shì shéi?
📌 Порядок: 他 + 是 + 谁？
💼 ПРОФЕССИЯ / ХАРАКТЕР
什么人 shénme rén что за человек
Спрашиваем профессию, характер, роль.
他是什么人？ Tā shì shénme rén?
📌 Порядок: 他 + 是 + 什么人？
🎯 ВЫБОР ИЗ ЛЮДЕЙ
哪个人 nǎ ge rén который из
Спрашиваем конкретного человека из группы.
哪个人是你弟弟？ Nǎ ge rén shì nǐ dìdi?
📌 Порядок: 哪个人 + 是 + 你 + 弟弟？
4. 什么时候 vs 几点
📅 ДАТА / ВРЕМЯ В ЦЕЛОМ
什么时候 shénme shíhou когда
Спрашиваем дату, день, время суток (не точное время).
你什么时候来？ Nǐ shénme shíhou lái?
📌 Порядок: 你 + 什么时候 + 来？
🕐 ТОЧНОЕ ВРЕМЯ (ЧАСЫ)
几点 jǐ diǎn который час
Спрашиваем точное время на часах.
现在几点？ Xiànzài jǐ diǎn?
📌 Порядок: 现在 + 几点？
5. 为什么 vs 怎么 (удивление)
❓ ПРИЧИНА
为什么 wèishénme почему
Спрашиваем причину, основание.
你为什么笑？ Nǐ wèishénme xiào?
📌 Порядок: 你 + 为什么 + 笑？
😲 УДИВЛЕНИЕ / НЕДОУМЕНИЕ
怎么 zěnme как так?
Выражает удивление, недоумение («как так?»).
你怎么来了？ Nǐ zěnme lái le?
📌 Порядок: 你 + 怎么 + 来了？
6. 多少 vs 几
💰 БОЛЬШИЕ ЧИСЛА / ЦЕНА
多少 duōshao сколько
Для чисел больше 10, для цены, количества.
这多少钱？ Zhè duōshao qián?
📌 Порядок: 这 + 多少 + 钱？
🔢 МАЛЕНЬКИЕ ЧИСЛА (< 10)
几 jǐ сколько
Для чисел от 1 до 10. ОБЯЗАТЕЛЬНО со счётным словом!
你几岁？ Nǐ jǐ suì?
📌 Порядок: 你 + 几 + 岁？
7. 哪儿 vs 哪里
🗣️ РАЗГОВОРНЫЙ
哪儿 nǎr где
Повседневный, разговорный вопрос о месте.
你在哪儿？ Nǐ zài nǎr?
📌 Порядок: 你 + 在 + 哪儿？
📖 НЕЙТРАЛЬНЫЙ / ВЕЖЛИВЫЙ
哪里 nǎlǐ где / откуда
Более формальный, вежливый вариант.
你是哪里人？ Nǐ shì nǎlǐ rén?
📌 Порядок: 你 + 是 + 哪里 + 人？
8. 干什么 ЧТО ДЕЛАТЬ?
🔨 ЧТО ДЕЛАТЬ? / ЗАЧЕМ?
干什么 gàn shénme что делать
Спрашиваем о действии, цели, занятии.
你在干什么？ Nǐ zài gàn shénme?
📌 Порядок: 你 + 在 + 干什么？
📐 ПРАВИЛА ПОРЯДКА СЛОВ В ВОПРОСАХ

В китайском языке порядок слов в вопросах почти не меняется — вопросительное слово ставится на то же место, где должен быть ответ!

Подлежащее + Вопрос + Сказуемое
Для вопросов к действию
你怎么去？
Подлежащее + Сказуемое + Вопрос
Для вопросов к объекту/месту
你是哪里人？
Вопрос + Подлежащее + Сказуемое
Для вопросов с выбором
哪本书是你的？
💡 Золотое правило: Вопросительное слово всегда стоит на месте ответа!
Ответ: 我坐地铁去。 → Вопрос: 你怎么去？
🔢 СЧЁТНЫЕ СЛОВА (счётчики)
Что такое счётные слова? В китайском языке каждый предмет имеет свой счётчик.
С 什么 счётные слова НЕ НУЖНЫ. С 哪, 几, 这, 那 — ОБЯЗАТЕЛЬНЫ!
📌 Порядок: Вопрос + СЧЁТНОЕ + Существительное
Пример: 哪本书？
